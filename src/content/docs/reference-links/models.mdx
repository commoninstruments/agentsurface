This is the canonical list of model IDs Agent Surface uses in examples. Every model-ID string in the docs and in the template kits must appear here. `scripts/check-docs-integrity.mjs` parses this page's code-formatted IDs into an allowlist and flags any model-ID-shaped string elsewhere in the corpus that is not on it. Update this page first, then run the sweep.

Verified against provider documentation on 2026-07-06.

## Anthropic

| Model     | API ID                      | Role                | Notes                                                                               |
| --------- | --------------------------- | ------------------- | ----------------------------------------------------------------------------------- |
| Fable 5   | `claude-fable-5`            | Flagship            | Highest-capability model; deep reasoning and long tasks.                            |
| Opus 4.8  | `claude-opus-4-8`           | Recommended default | Best general default for complex agent work.                                        |
| Sonnet 5  | `claude-sonnet-5`           | Balanced            | Fast iteration and everyday agent loops.                                            |
| Haiku 4.5 | `claude-haiku-4-5-20251001` | Fast / low cost     | Cheap evals, classification, high-volume calls. Dateless alias: `claude-haiku-4-5`. |

**Retired 2026-06-15:** Opus 4 and Sonnet 4. Older pins from that era — Opus 4.7 and Sonnet 4.6 — are no longer valid; the model-ID check flags any that remain in examples. Replace flagship pins with `claude-opus-4-8` and balanced pins with `claude-sonnet-5`.

## OpenAI

| Model   | API ID    | Role       | Notes                                                                      |
| ------- | --------- | ---------- | -------------------------------------------------------------------------- |
| GPT-5.5 | `gpt-5.5` | Flagship   | Use where an example calls for OpenAI's most capable model.                |
| GPT-5.4 | `gpt-5.4` | Value tier | Cost-efficient default; correct wherever a cheaper OpenAI model is wanted. |

The o-series reasoning models (o1, o3, o4 family) were deprecated in June 2026, with removal scheduled for December 2026. Do not use them in examples.
